Product

Para adicionar o código de acompanhamento de produto, será necessário adicionar algumas informações, essas que serão enviadas para analise. O código de acompanhamento de produto pode ser adicionado em vitrines de produtos, páginas de detalhe de produto, entre outras páginas onde podem conter produtos.


Nome do campo Valor do campo Tipo do campo Obrigatório Descrição
window._appHarpy.harpyId [Seu HarpyID] String Sim harpyId é o código de acompanhamento do e-commerce, que pode ser gerado no cadastro do site.
window._appHarpy.event Product String Sim Tipo do módulo que será inicializado
window._appHarpy.hitType view ou click String Sim Tipo de evento que será acionado para a captura das informações
window._propsHarpy.product [Ver tabela de propriedades] Array Sim _propsHarpy são as propriedades que serão enviadas para cada evento acionado

Tabela de propriedades para o evento [Product]

Como o campo _propsHarpy.product é um array, então será necessário adicionar um objeto javascript com as informações do produto.

window._propsHarpy.product
Nome da chave do objeto Tipo do campo Obrigatório Descrição
id String Sim ID do produto
name String sim Nome do produto
price Currency Não Preço do produto
list String Não Nome da vitrine ou lista em que o produto se encontra
category String Não Nome da categoria do produto
variant String Não Variação do produto, como cor, voltagem, ou outra função.
brand String Não Marca do produto
position Integer Não Posição em que o produto se encontra na lista/vitrine
Exemplo de implementação:

// Product
<script src="//52.11.59.146/harpymetrics.min.js"></script>
<script>
window._appHarpy.event = 'Product'; // Tipo do módulo
window._appHarpy.harpyId = 'HID15b2330'; // HarpyId
window._appHarpy.hitType = 'Detail'; // Tipo de eventos

var products = [];
$(myProducts).each(function(i,e) {
if(e.pageCategory) {
var prod = {};
prod.id = e.productId;
prod.name = e.productName;
prod.price = e.productPriceTo;
prod.list = e.list;
prod.category = e.productCategoryName;
prod.brand = e.brand;
prod.variant = e.skuName;
prod.position = '0';
products.push(prod);
}
});

window._propsHarpy.product = products; // Onde será enviado as propriedades
Harpy.module('product'); // Onde será inicializado o módulo de produto

</script>
