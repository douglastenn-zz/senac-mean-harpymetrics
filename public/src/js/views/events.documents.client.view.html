Eventos

Eventos podem ser considerados como uma interação, que resulta nos dados que estão sendo enviados para o harpymetrics. Os tipos de eventos conseguem incluir acesso de páginas, rastreamento dos passos do usuário, acompanhamento no checkout, entre outros.


Cada vez que um código de acompanhamento é adicionado um evento de comportamento do usuário é acionado. Por exemplo, o usuário carrega a página de produto, isso dispara uma ação de detail que seria para verificar os acessos a determinadas páginas do site. Após isso o harpymetrics registra essa atividade com as demais informações capturadas na página.


Abaixo segue a lista dos eventos:

Nome do campo Valor do campo Descrição Módulos (onde podem ser utilizados)
hitType View Evento para visualização. Vitrines de produtos ou promoções internas
hitType Click Evento para captura de cliques. Vitrines de produtos ou promoções internas
hitType Send Atividade para rastreamento dos envios de formulários Todos os formulários
hitType Detail Evento para captura das páginas acessadas. Todas as páginas
hitType Add Cart Rastreamento das ações de adicionar ao carrinho Ação de adicionar produto do carrinho
hitType Remove Cart Rastreamento das ações de remover do carrinho Ação de remover produto do carrinho
hitType Purchase Eventos para as conversões Finalização do processo de checkout
hitType Checkout Eventos de acompanhamento do checkout Checkout
