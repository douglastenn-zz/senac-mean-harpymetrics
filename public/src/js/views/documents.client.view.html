Eventos

Para o HarpyMetrics iniciar as suas atividades é necessário utilizar o script fornecido pelo harpymetrics e adicionar algumas tags no site que será monitorado. As páginas principais onde devem ser inseridos os scripts são: Home, Search, Category, Product, Checkout.

Ver mais detalhes

Home

Na homepage o harpymetrics consegue monitor os eventos de view (visualização) ou click (clique) para produtos e promoções internas, conforme a necessidade.

Ver mais detalhes

Search

Na página de busca o harpymetrics pode relatar a quantidade total de buscas e quais foram os termos mais buscados e alinhar esses resultados com as informações de outros relatórios.

Ver mais detalhes

Category

Com o script de categoria podemos visualizar quais foram as categorias mais acessadas, além de encontrar padrões de navegação que podem levar a uma categoria específica.

Ver mais detalhes

Product

Produto é o núcleo do e-commerce, os relatórios do harpymetrics ajuda a criar novas percepções de como os usuários estão navegando, assim, podendo criar uma boa estrátegia de venda.

Ver mais detalhes

Checkout

Cada passo de checkout pode ser monitorado pelo harpymetrics, assim pode ser descrito o desempenho do carrinho de compras e pedidos confirmados, além de averiguar os dados de navegação a cada passo do usuário.
