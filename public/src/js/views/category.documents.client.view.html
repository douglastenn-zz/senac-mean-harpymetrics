Category

Para adicionar o código de acompanhamento de página de categoria, será necessário adicionar algumas informações, essas que serão enviadas para analise.


Nome do campo Valor do campo Tipo do campo Obrigatório Descrição
window._appHarpy.harpyId [Seu HarpyID] String Sim harpyId é o código de acompanhamento do e-commerce, que pode ser gerado no cadastro do site.
window._appHarpy.event Category String Sim Tipo do módulo que será inicializado
window._appHarpy.hitType detail String Sim Tipo de evento que será acionado para a captura das informações
window._propsHarpy.category [Ver tabela de propriedades] Array Sim _propsHarpy são as propriedades que serão enviadas para cada evento acionado

Tabela de propriedades para o evento [Category]

Como o campo _propsHarpy.category é um array, então será necessário adicionar um objeto javascript com as informações da categoria.

window._propsHarpy.category
Nome do campo Tipo do campo Obrigatório Descrição
id String Sim ID da categoria de produtos
name String Não Nome da categoria de produtos
Exemplo de implementação:

// Category
<script src="//52.11.59.146/harpymetrics.min.js"></script>
<script>
window._appHarpy.event = 'Category'; // Tipo do módulo
window._appHarpy.harpyId = 'HID15b2330'; // HarpyId
window._appHarpy.hitType = 'Detail'; // Tipo de evento

var category = [];
$(myCategories).each(function(i,e) {

if(e.pageCategory) {
var cat = {};
cat.id = e.siteSearchResults;
cat.name = e.siteSearchTerm;
category.push(cat);
}

});

window._propsHarpy.category = category; // Onde será enviado as propriedades
Harpy.module('category'); // Onde será inicializado o módulo de categoria
</script>
