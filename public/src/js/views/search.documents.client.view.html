Search - Busca

Para adicionar o código de acompanhamento na busca, será necessário adicionar algumas informações básicas no código.


Nome do campo Valor do campo Tipo do campo Obrigatório Descrição
window._appHarpy.harpyId [Seu HarpyID] String Sim harpyId é o código de acompanhamento do e-commerce, que pode ser gerado no cadastro do site.
window._appHarpy.event Search String Sim Tipo do módulo que será inicializado
window._appHarpy.harpyId [Seu HarpyID] String Sim harpyId é o código de acompanhamento do e-commerce, que pode ser gerado no cadastro do site.
window._propsHarpy.search [Ver tabela de propriedades] Array Sim _propsHarpy são as propriedades que serão enviadas para cada evento acionado
Tabela de propriedades para o evento [Search]

Como o campo _propsHarpy.search é um array, então será necessário adicionar um objeto javascript com as informações de busca.


window._propsHarpy.search
Nome da chave do objeto Tipo do campo Obrigatório Descrição
searchTerm String Sim Campo que será adicionado o termo buscado pelo usuário.
totalResult Integer Sim Total de itens encontrados pela busca do usuário
Exemplo de implementação:

// Search
<script src="//52.11.59.146/harpymetrics.min.js"></script>
<script>
window._appHarpy.event = 'Search'; // Tipo do módulo
window._appHarpy.harpyId = 'HID15b2330'; // HarpyId
window._appHarpy.hitType = 'Detail'; // Tipo de evento

var searchArr = [];
$(mySearch).each(function(i,e) {

if(e.siteSearchResults) {
var search = {};
search.searchTerm = e.siteSearchTerm;
search.totalResult = e.siteSearchResults;
searchArr.push(search);
}

});

window._propsHarpy.search = searchArr; // Onde será enviado as propriedades
Harpy.module('search'); // Onde será inicializado o módulo para a busca
</script>
