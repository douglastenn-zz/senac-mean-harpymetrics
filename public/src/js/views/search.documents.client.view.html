Para adicionar o código de acompanhamento na busca, será necessário adicionar algumas informações básicas no código.


Nome do campo Valor do campo Tipo do campo Obrigatório Descrição
window._appHarpy.harpyId [Seu HarpyID] String Sim harpyId é o código de acompanhamento do e-commerce, que pode ser gerado no cadastro do site.
window._appHarpy.event Search String Sim Tipo do módulo que será inicializado

Como o campo _propsHarpy.search é um array, então será necessário adicionar um objeto javascript com as informações de busca.
